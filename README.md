## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | The unique URL of the company profile on VentureRadar. |
| `location` | The geographical location of the company. |
| `name` | The official name of the company. |
| `country_code` | The ISO country code representing the company's location. |
| `ownership` | The type of ownership such as private or public company. |
| `score` | The company score indicating its overall standing or performance. |
| `auto_analyst_score` | An automated score reflecting the analyst's evaluation. |
| `website_popularity` | An indicator of the company's website popularity on the internet. |
| `email` | The contact email address for the company, partially obscured for privacy. |
| `keywords` | Keywords associated with the company's industry or sector. |
| `linkedin` | The LinkedIn profile URL of the company. |
| `twitter` | The Twitter handle of the company. |
| `founded` | The year the company was founded. |
| `title` | The title or designation of the company. |
| `description` | A brief description of the company's operations and offerings. |
| `description_source` | The source from which the company description was derived. |
| `website` | The official website URL of the company. |
| `website_popularity_graph` | Graphical data representing the company's website popularity trends. |
| `similar` | A list of companies similar in nature or operation. |
| `sub_scores` | Detailed sub-scores evaluating various aspects of the company. |
| `hostorical_profiles` | Historical profile data of the company. |
| `areas_of_focus` | Key areas or industries where the company focuses its operations. |
| `cards` | Additional informational cards related to the company. |
| `funding_signals` | Indicators of the company's funding activities or needs. |
| `employee_satisfaction` | Metrics or scores reflecting employee satisfaction within the company. |
| `company_id` | A unique identifier assigned to the company within the dataset. |
