# VentureRadar 회사 정보 데이터셋 샘플

<h2>1,000개 레코드의 샘플 데이터셋</h2>

이 **VentureRadar 회사 정보 데이터셋 샘플**에는 **1,000개 레코드**가 포함되어 있으며, **Bright Data API**를 사용하여 추출되었습니다.

## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | VentureRadar에서 회사 프로필의 고유 URL입니다. |
| `location` | 회사의 지리적 위치입니다. |
| `name` | 회사의 공식 명칭입니다. |
| `country_code` | 회사 위치를 나타내는 ISO 국가 코드입니다. |
| `ownership` | 비상장 또는 상장 기업과 같은 소유 형태입니다. |
| `score` | 전반적인 위상 또는 성과를 나타내는 회사 점수입니다. |
| `auto_analyst_score` | 분석가의 평가를 반영하는 자동화된 점수입니다. |
| `website_popularity` | 인터넷상에서 회사 웹사이트의 인기도를 나타내는 지표입니다. |
| `email` | 개인정보 보호를 위해 일부가 마스킹된 회사 연락 이메일 주소입니다. |
| `keywords` | 회사의 산업 또는 섹터와 연관된 키워드입니다. |
| `linkedin` | 회사의 LinkedIn 프로필 URL입니다. |
| `twitter` | 회사의 Twitter 핸들입니다. |
| `founded` | 회사 설립 연도입니다. |
| `title` | 회사의 타이틀 또는 명칭입니다. |
| `description` | 회사의 운영 및 제공 서비스에 대한 간략한 설명입니다. |
| `description_source` | 회사 설명이 도출된 출처입니다. |
| `website` | 회사의 공식 웹사이트 URL입니다. |
| `website_popularity_graph` | 회사 웹사이트 인기도 추세를 나타내는 그래프 데이터입니다. |
| `similar` | 성격 또는 운영이 유사한 회사 목록입니다. |
| `sub_scores` | 회사의 다양한 측면을 평가하는 상세 하위 점수입니다. |
| `hostorical_profiles` | 회사의 과거 프로필 데이터입니다. |
| `areas_of_focus` | 회사가 운영에 집중하는 핵심 영역 또는 산업입니다. |
| `cards` | 회사와 관련된 추가 정보 카드입니다. |
| `funding_signals` | 회사의 자금 조달 활동 또는 필요를 나타내는 지표입니다. |
| `employee_satisfaction` | 회사 내 직원 만족도를 반영하는 지표 또는 점수입니다. |
| `company_id` | 데이터셋 내에서 회사에 부여된 고유 식별자입니다. |


## Use Cases

### 1. Market Analysis
이 데이터셋을 활용하여 심층적인 시장 분석을 수행하고 다양한 섹터 전반에서 새로운 비즈니스 트렌드를 파악할 수 있습니다.

### 2. Competitive Intelligence
데이터셋을 사용하여 경쟁사의 강점과 약점에 대한 인사이트를 확보함으로써 전략적 비즈니스 의사결정에 도움을 받을 수 있습니다.

### 3. Investment Opportunities
회사 점수 및 산업 집중 영역을 분석하여 잠재적 투자 대상으로 유망한 스타트업과 회사를 식별할 수 있습니다.
